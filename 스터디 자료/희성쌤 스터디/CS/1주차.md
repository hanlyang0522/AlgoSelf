# 자료구조

## 배열(Array)
- 메모리 상에서 데이터가 연속적
- 크기는 고정. 변경될 수 없음
- O(1): n번째 데이터 접근, 변경
- O(n): x라는 데이터가 있는지 탐색

## Dynamic Array(List, ArrayList, Vector)
- 메모리 상에서 데이터가 연속적으로 연결
- 크기는 동적 변경이 가능하나, 근본적으로 배열 기반 구현
- **사이즈를 늘리는 로직상 메모리 낭비 발생이 가능**
- O(1): n번째 데이터 접근, 변경
- O(n): x라는 데이터가 있는지 탐색
- O(n): 데이터 삽입/삭제
- Amortized O(1): 리스트 끝에 데이터 삽입/삭제

## Linked List
- 메모리 상에선 불연속이나, 논리상에선 연속
- Value, 다음 메모리 주소를 node라는 이름의 쌍으로 저장
